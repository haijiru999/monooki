# クラス名

| 名前      | 説明                                 |
| --------- | ------------------------------------ |
| Connector | 外部と通信する                       |
| Service   | よくよく考えたら意味がよくわからない |

## メソッド名

| 名前    | 説明           |
| ------- | -------------- |
| request | 通信で要求する |

- init - 初期化する（1回限り）
- show - 表示する（その都度）
- dispose - 破棄する（initの逆）
